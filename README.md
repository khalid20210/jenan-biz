README — ملفات المشروع (المرفوعات الأخيرة)

ملف هذا يشرح بسرعة الملفات التي تم رفعها مؤخراً إلى المستودع.

قائمة الملفات المرفوعة ووصف موجز لكلٍ منها:

- `api/` : مجلد الخادم الخلفي (FastAPI). يحتوي على `services.py` الذي يشغّل API ونقاط النهاية، وملف `requirements.txt` لكل المتطلبات.
- `assets/` : ملفات الواجهة الأمامية (CSS, JS, images). تضم الوحدات النمطية مثل `business-analyzer.js` لتحليل المشاريع.
- `config/` : إعدادات التطبيق وملفات التهيئة `app-config.js`.
- `auth.html` : صفحة الدخول/التسجيل (واجهة المستخدم الأساسية).
- `dashboard.html` : لوحة التحكم الرئيسية للمستخدم.
- `pricing.html` : صفحة التسعير والاشتراك.
- `report-preview.html` : معاينة تقارير الجدوى.
- `companies.html` : صفحة دليل الشركات.
- `logo.png`, `logo.jpg` و`شعار جنان بيز.jpg` : شعارات وصور العلامة.
- `.gitignore` : ملفات ومجلدات المستبعدة من المستودع.
- `setup_jenan.py` : سكربت إعداد/تهيئة المشاريع وقاعدة البيانات.

ملاحظات مهمة:
- نقطة الدخول للخادم: `api/services.py` — يمكن تشغيله محلياً عبر:

```powershell
cd "c:\Users\JEN21\OneDrive\سطح المكتب\المدونه\Jenan-blog"
& ".venv\Scripts\python.exe" -m uvicorn api.services:app --host 0.0.0.0 --port 8000 --reload
```

- ملف `api/requirements.txt` يحتوي على تبعيات الإنتاج والتطوير اللازمة.
- أي تغييرات أو إضافات أريد أن أرفعها مباشرةً سأقوم بـ `git add` و`git commit` و`git push` بحسب صلاحيات الريموت.

إذا تريَد توضيح أو تفصيل أكثر عن أي ملف أو نقطة تشغيل، أخبرني وسأحدّث هذا README.
